## Search & Products (Combed 20s)

- Home search: header button Search (role=button name=Search) opens overlay; input placeholder="Search". Suggestions list products (e.g. COTTON COMBED 20S …) but suggestion link clicks often do not navigate — prefer catalog URLs.
- Product catalog URL pattern: `/shop/fabrics/{fabric}` and `/shop/fabrics/{fabric}/{count}` and SKU `/shop/fabrics/{fabric}/{count}/{slug}`.
- Cotton Combed listing: https://knitto.co.id/shop/fabrics/cotton-combed
- Combed 20s family: https://knitto.co.id/shop/fabrics/cotton-combed/20s — title "Cotton Combed 20s | Toko Bahan Kaos Knitto"; price range 105.000 - 129.000 / kg (roll); 32 color variants.
- Example SKU: https://knitto.co.id/shop/fabrics/cotton-combed/20s/combed-20s-abu-sedang (BARU, 113.000 - 114.000 / kg).
- Store name: Knitto / Toko Bahan Kaos Knitto. Do not click Beli/checkout for search-only tasks.
- Nav Belanja /shop/fabrics path works; /belanja redirects to home.
