- Pilih menu "Warna Kain (Detail Kain)"
- cari nama kain web dengan value {nama_kain_web} lalu tekan Enter
- cari warna kain dengan value {warna_kain} lalu tekan Enter
- cari kategori kain dengan value {kategori_kain} lalu tekan Enter
- klik edit (icon pensil) pada row
- Klik "Upload Foto" lalu masukan file {gambar_foto_produk} bisa single/multiple
- klik simpan
